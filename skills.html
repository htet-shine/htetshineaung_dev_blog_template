My Skills

Web Development

Laravel

Laravel is a php framework that lets you create web apps in a Model, View, Controller pattern. A lot of security features are already included in the framework such as SQL Injection Protection, cross site request forgery, etc. Availability of very good plugins also speed up the development.
I have started 2 applications/projects with laravel so far. You can find developement process, installation, and user guides in my "Posts" navigation menu.

Level: Beginner

Wordpress

Wordpress is the most popular Content Management System in the world. Its easy to use for users without technical knowledge and it also allow developers who want to go further by creating custom plugins and themes.
This site is created in wordpress with my own theme which include custom post types, taxonomies and WP queries. For clients who want wordpress as a backend for their site. I have the base knowledge to be part of wordpress development.

Level: Beginner

PHP

PHP is server side scripting language. This means that PHP is responsible for putting logic into websites which allow for dynamic content.
I have done a couple of small little projects while taking PHP classes. But all of those little elements and a lot more are added to an e-Commerce project. And it is written in procedural PHP which is easier to learn for me as a beginner. You can find the project in "Posts" menu.

Level: Intermediate

HTML, CSS & JS

HTML is a standard language to structure documents that can be displayed on a web browser. CSS add styling to documents and responsiveness. Javascript makes elements on a website interactive by DOM manipulation and dynamic by using ajax.
In my PHP e-Commerce project, I've use jQuery library for DOM manipulation and jQuery ajax for database queries without reloading te pages.

Level: Intermediate

Others

Aodbe Photoshop

Photoshop is probably the most widely used professional image editing software.
I've learned Photoshop as a hobby. Now it is helping me in web developemt. Level: Intermediate

Adobe Premiere Pro

Adobe Premiere Pro is also a very popular video editing software.
Again I've learned this as a hobby. By no means I'm a professional video editor but I can cut and join multiple clips, add transitions, titles, graphics and background music, adjust colors, brightness and sharpness of a video. One of the main reason of using Premiere Pro is that it can dynamically linked to a Adobe After Effects Project and changes are synced. This save a lot of time by eliminating the need to render from After Effects.

Level: Intermediate

Adobe After Effects

Adobe After Effects is also a very popular special effects software.
You can add titles and simple graphics in Premiere Pro but if you want a little bit more complex animations, After Effects is the way to go. And it works seamlessly with Photoshop artboards and Premiere Pro. I haven't done 3D animations but I can animate 2D objects by using layers, keyframes, masks and track matte. After affects works well with photoshop. I can create a scene in photoshop first to organize everything in one file and bring it to After Effects.

Level: Intermediate
